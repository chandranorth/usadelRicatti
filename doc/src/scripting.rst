Scripting
=========

This section contains several example calculations for simple SNS
junctions and for more complicated structures.

Before starting, if you are not familiar with the Python language, you
may save some time later on by glancing through the following document:

- Python tutorial: `Part 1 <http://docs.python.org/dev/tutorial/introduction.html>`__, 
  `Part 2 <http://docs.python.org/dev/tutorial/controlflow.html>`__.

For information about arrays and scientific routines in Python, see

- http://docs.scipy.org/doc/numpy/reference/
